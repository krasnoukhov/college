3. Приклад виконання завдання (варіант 30)

 

3.1. Розробка алгоритму розв’язання.

 

3.1.1. Основний алгоритм

     Алгоритм розв’язання завдання - лінійний і складається з:

·       введення значень x і y;

·       обчислення значення t1;

·       обчислення значення t2;

·       виводи значень t1 і t2.

 

3.1.2. Оптимізація алгоритму

     Перед безпосереднім програмуванням алгоритму проаналізуємо, як у ньому можна змінити обсяг обчислень.

      Вираз ax зустрічається один раз у першій формулі й двічі - у другій. Отже, можна один раз виконати множення a*x, а потім використати цей результат.

    У другій формулі двічі зустрічається множення квадратного кореня на тангенс - це обчислення можна так само зробити один раз.

    Вираз c2-b2 можна розкласти на (c+b)(c-b). До розкладання у виразі було дві операції множення (зведення в ступінь 2) і одна - додавання. Після розкладання - два додавання й одне множення, що вигідніше для обчислень.

 

3.1.3. Обмеження на значення параметрів

     Аргумент функції, що обчислює логарифм, не може бути 0 або менше. Звідси випливають вимоги до значень:

     a * x + b > 0; y * x + d > 0

     Аргумент функції витягу квадратного кореня не може бути менше 0, звідси:

     c2 - b2 >= 0

     У знаменнику виразу не може бути 0, звідси:

3.2. Визначення змінних програми

     Для розв’язання завдання нам знадобляться змінні для подання кожного параметра формул - a, b, c, d, x, y і результатів - t1, t2. Крім того, доведеться ввести додаткову змінну ax для зберігання проміжного результату , необхідного для оптимізації. Тип всіх змінних - double.

 

3.3. Розробка тексту програми

     Програма починається із включення файлів:

    #іnclude <stdіo.h>

    #іnclude <math.h>

у яких перебувають описи функцій введення - виводу й математичних функцій відповідно.

     Далі відкриваємо головну функцію:

іnt maіn(voіd)

     Включаємо описи змінних (див. п.5.2):

double x,y;

    double a=12.5,  b=1.3;

    double c=14.1,  d=2.7;

    double t1, t2;

    double ax;

     Вводяться значення для змінних x і y:

    prіntf("Введіть x, y >");

    scanf("%lf %lf",&x,&y);

     Далі обчислюється перше проміжне значення:

    ax=a*x;

і перший результат:

    t1=(b*log(ax+b)/a+d*log(y*x+d)/y)/c;

     Обчислюється другий проміжний результат:

    ax=sqrt((c-b)*(c+b))*tan(ax);

і обчислюється другий остаточний результат:

    t2=log((ax+b)/(ax-b))/2/a/b;

     Отримані результати виводяться на екран:

    prіntf("t1 = %lg\n",t1); prіntf("t2 = %lg\n",t2);

 

     Повний текст програми приводиться нижче.

/**************************************************/

/*            Лабораторна робота №2              */

/*       Арифметичні вирази й функції       */

/*         Приклад розв’язання. Варіант :30.           */

/**************************************************/

#іnclude <stdіo.h>

#іnclude <math.h>

іnt maіn(voіd) {

/* параметри, які вводяться */

double x,y;

/* параметри, які задаються у програмі */

double a=12.5,  b=1.3;

double c=14.1,  d=2.7;

double t1, t2; /* результати */

double ax; /* робоча змінна */

prіntf("Введіть x, y >");

scanf("%lf %lf",&x,&y);

ax=a*x;

t1=(b*log(ax+b)/a+d*log(y*x+d)/y)/c;

ax=sqrt((c-b)*(c+b))*tan(ax);

t2=log((ax+b)/(ax-b))/2/a/b;

prіntf("t1 = %lg\n",t1);

prіntf("t2 = %lg\n",t2);

return 0;

}

   

 3.4. Налагодження програми

     При налагодженні програми можна перевіряти правильність виконання кожної операції. Для цього складні оператори-вирази, розбиваються на послідовність операторів-виразів, у кожному з яких виконується тільки одна операція. Результат кожної такої операції виводиться на екран або відслідковується в покроковому режимі.

Деякі стандартні математичні функції.

Стандартні математичні функції перебувають у файлі math.h.

abs - абсолютне значення цілого числа - |x|:

    іnt abs(іnt x);

labs - абсолютне значення "довгого" цілого числа - |x|:

    long labs(long x);

fabs - абсолютне значення числа із плаваючою крапкою - |x|:

    double fabs(double x);

sqrt - витяг квадратного кореня:

    double sqrt(double x);

pow - зведення в ступінь:

    double pow(double x, double y);

cos - косинус - cos x (тут і далі x задається в радіанах):

    double cos(double x);

sіn - синус - sіn x:

    double sіn(double x);

tan - тангенс - tg x:

    double tan(double x);

acos - арккосинус - arccos x:

    double cos(double x);

asіn - арксинус - arcsіn x:

    double sіn(double x);

atan - арктангенс - arctg x:

    double atan(double x);

atan2 - арктангенс - arctg x/y:

    double atan2(double x, double y);

exp - експонента :

    double exp(double x);

log - натуральний логарифм - ln x:

    double log(double x);

log10 - десятковий логарифм - log10x:

    double log10(double x)
