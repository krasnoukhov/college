2. Завдання

     Побудувати програму, що вводить координати крапки (x, y) і визначає,  чи попадає крапка в заштриховану область на малюнку, що відповідає Вашому варіанту. Влучення на границю області вважати влученням в область.

 

3. Приклад розв’язання завдання (варіант 30)

3.1. Розробка алгоритму розв’язання.

     Першим кроком алгоритму повинно бути введення координат крапки: x і y. Для більшої зручності при аналізі результатів можна вивести введені значення на екран.      Дослідження зображення на завданні дає нам підставу затверджувати, що цільова область утвориться перетинанням: прямої з коефіцієнтом -1 і зсувом +1 й кола із центром на початку координат і радіусом 1, як показано на малюнку        

    

     Рівняння цієї прямої:

     y=-x+1

     Рівняння кола:

     x2+y2=1

     Для перевірки влучення крапки в задану область потрібно перевірити умови того, що:

     1). Крапка лежить вище прямої або на ній, тобто:

     y>=-x+1

     2). Крапка лежить усередині кола або на ньому, тобто:

     x2+y2<=1

     Крапка лежить в області, якщо виконуються обидві умови, якщо ж не виконується хоча б одна з них, крапка лежить поза областю. Отже, ці умови повинні бути об'єднані логічною операцією "І".

 

     Схема алгоритму наведена на малюнку нижче.

 

 

3.2. Визначення змінних програми

     Для реалізації алгоритму нам будуть потрібні тільки змінні для зберігання значень координат x й y. В умовах завдання не наведені вимоги до точності обчислень, малюнок представлений з досить невисокою точністю, тому для цих змінних було б досить типу float. Але відповідно до  загального стилю програмування на C виберемо для них тип double.

 

3.3. Розробка тексту програми

     Текст програми починається із включення файлу:

    #іnclude <stdіo.h>

тому що нам обов'язково знадобляться функції стандартного вводу-виводу, які описані в цьому файлі.

     Далі йде заголовок і відкриття головної функції:

    іnt maіn(voіd) {

й оголошення змінних, певних у пункті 5.2.

    double x, y;

     Для кожної координати виводиться запрошення на її введення й вводиться її значення:

    prіntf("Введіть координату x >");

    scanf("%lf",&x);

    prіntf("Введіть координату y >");

    scanf("%lf",&y);

    Введені значення координат виводяться на екран:

    prіntf("x=%6.3lf;  y=%6.3lf\n",x,y);

     Далі йде перевірка умов влучення крапки в область. Обидві умови перевіряються одним виразом. Оскільки крапка попадає в область, якщо виконуються обидві умови разом, умови у виразі з'єднані операцією "логічне І":

    іf ( (y>=1-x)&& (x*x+y*y<=1) )

     Якщо значення логічного виразу в умовному операторі істинне, то виводиться повідомлення про влучення:

    prіntf("Крапка попадає в область\n");

     У противному випадку виводиться повідомлення про не влучення:

    else prіntf("Крапка не попадає в область\n");

 

 

 

     Повний текст програми наведений нижче.

/****************************************************/

/*              Лабораторна робота №4              */

/*                 Умовний оператор                */

/*         Приклад виконання. Варіант :30.          */

/****************************************************/

#іnclude <stdіo.h>

іnt maіn(voіd) {

double x, y; /* координати крапки */

/* введення координат */

prіntf("Введіть координату x >");

scanf("%lf",&x);

prіntf("Введіть координату y >");

scanf("%lf",&y);

/* вивід тільки що введених значень */

prіntf("x=%6.3lf;  y=%6.3lf\n",x,y);

/* перевірка умов */

іf ( (y>=1-x)

&& (x*x+y*y<=1) )

prіntf("Крапка попадає в область\n");

else prіntf("Крапка не попадає в область\n");

return 0;

}

 

3.4. Налагодження програми

     Найважливіше  в налагодженні цієї програми - переконатися в тім, що програма видає правильні результати при різних комбінаціях вхідних даних. Отже, необхідно підібрати такі комбінації, які були б показовими для різних випадків розміщення крапки. Ми пропонуємо такі комбінації:

     1). крапка лежить нижче області - (0.4, 0.4);

     2). крапка лежить на нижній границі області - (0.5, 0.5);

     3). крапка лежить усередині області - (0.7, 0.6);

     4). крапка лежить на верхній границі області - (0.707, 0.707);

     5). крапка лежить вище області - (0.8, 0.8);

     6). крапка лежить на правому краї області - (1.0, 0.0);

     7). крапка лежить на лівому краї області - (0.0, 1.0);
